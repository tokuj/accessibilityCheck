# フロントエンドアクセシビリティチェックアプリ ユーザー体験

このアプリケーションは、ウェブサイトのアクセシビリティを評価し、改善点を提案することを目的としています。ユーザー体験（UX）を向上させるために、以下の主要な機能を保有しています。

## アクセシビリティ分析機能

アプリケーションを開くと下記のスクリーンショットの様にURLを入力することが出来るインタフェースがあります。ここに評価したいウェブサイトのURLを入力し、「分析開始」ボタンをクリックします。
![インプット画面](/docs/img/input.png)

分析が完了すると、以下のような詳細なレポートが表示されます。このレポートには、アクセシビリティの問題点、改善提案、及び各問題の重要度が含まれています。
![レポート画面](/docs/img/report.png)
