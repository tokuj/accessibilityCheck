# TODO フォルダ

## 概要

このフォルダにはプロジェクトの計画ファイルと作業ログが格納されています。

## 構造

```
TODO/
├── WORKLOG/              # 作業ログ
│   ├── index.md          # WORKLOGインデックス
│   └── YYYY-MM-DD.md     # 日次ログファイル
├── PLAN_*.md             # 実装計画ファイル
└── README.md             # このファイル
```

## ファイルの説明

- **WORKLOG/**: 日次の作業履歴を記録
- **PLAN_*.md**: 各機能・タスクの実装計画
