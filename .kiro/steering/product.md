# Product Overview

Webアクセシビリティを自動でテスト・分析するツール。開発者やQAエンジニアが、WCAG準拠状況を効率的に確認できる。

## Core Capabilities

1. **マルチツール分析**: axe-core、Pa11y、Lighthouseの3つのエンジンを統合し、網羅的なアクセシビリティチェックを実施
2. **CLIテスト**: Playwright + axe-coreによる自動テストスイート
3. **Web UI**: URLを入力して即座に分析結果を可視化
4. **認証サポート**: Basic認証、フォーム認証、セッション認証に対応

## Target Use Cases

- **定期的なアクセシビリティ監査**: サイト全体のWCAG準拠状況を定期チェック
- **CI/CDパイプライン統合**: リリース前のアクセシビリティ回帰テスト
- **開発中の品質確認**: 新規ページ・機能のアクセシビリティ検証

## Value Proposition

- 単一ツールでは見逃しがちな問題を複数エンジンで検出
- WCAG 2.0/2.1 Level A/AA基準に対応
- 違反箇所の具体的な要素と改善ヘルプを提供
